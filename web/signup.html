Create a Mesh

Set up a new Lion–Bunny mesh on this relay. You'll get an invite code to share with your Bunny.

Paste the RSA public key from Lion's Share → Advanced → Export Key. This key signs your orders — the relay never sees your private key.

The invite code expires in 24 hours and can only be used once. Your Bunny enters it in Bunny Tasker → Join Mesh along with this relay's URL. Or scan the QR code above.
