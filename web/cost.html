Cost of Running

What it costs to operate this relay, in full transparency.

Monthly Server Cost

$0

This relay runs on homelab hardware. The only ongoing cost is electricity, which is negligible (a single-board computer drawing ~15W).

Vault blobs are opaque ciphertext. A typical mesh generates ~100 blobs/day (one every ~15 seconds during active use). At that rate, 100 MB lasts over two years.

What You're Trusting

The relay stores encrypted blobs only. It verifies RSA signatures (to prevent tampering) but cannot read any order content — lock commands, paywall amounts, messages, coordinates, or anything else.

The relay source is public and auditable. The /version endpoint returns a SHA-256 hash of the running code and a git commit. You can verify at any time that what's deployed matches what's published.

Full cryptographic design: VAULT-DESIGN.md

Why It's Free

There's nothing to monetize. The relay sees ciphertext, not content. There are no ads, no analytics, no accounts beyond the mesh registration. The operator pays for hardware and electricity because the project exists to be useful, not profitable.

If the relay ever goes offline, your mesh keeps working — devices gossip directly over LAN and Tailscale. You can also self-host the relay on your own hardware.

Self-Host Instead

Don't trust this relay? Run your own. The server is a single Python file with one dependency (cryptography). Runs on any Linux box, Raspberry Pi, or $5/mo VPS.
