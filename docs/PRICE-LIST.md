# The Collar — Price List

All penalties are added to the paywall. Only Lion can clear the paywall.
Compound interest applies while locked (10%/hr Bronze/none, 5%/hr Silver, 0% Gold).

## Subscriptions (Weekly)

| Tier | Cost | Perks |
|------|------|-------|
| Bronze | $25/wk | Base tier |
| Silver | $35/wk | 5%/hr interest (vs 10%), check-in reminders |
| Gold | $50/wk | 0% interest, 1 free unlock/month |
